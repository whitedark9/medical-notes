# Common Postoperative Complications

## 1. Immediate (0-24h Post-Op)

| Complication               | Cause                                                | Presentation                  | Management                                       |
| -------------------------- | ---------------------------------------------------- | ----------------------------- | ------------------------------------------------ |
| **Airway Obstruction**     | Tongue relaxation, laryngospasm, edema               | Stridor, desaturation         | Jaw thrust, suction, intubation if needed        |
| **Hypoxia**                | Atelectasis, aspiration, pulmonary edema             | Dyspnea, low O₂ sat           | Oxygen, chest physiotherapy                      |
| **Bleeding/Hemorrhage**    | Surgical site bleeding, coagulopathy                 | Hypotension, tachycardia      | Fluid resuscitation, transfusion, re-exploration |
| **Hypotension/Shock**      | Hemorrhage, sepsis, cardiogenic shock                | Low BP, altered mental status | IV fluids, pressors, treat cause                 |
| **Malignant Hyperthermia** | Reaction to anesthetics (succinylcholine, halothane) | High fever, muscle rigidity   | Dantrolene, cooling measures                     |

## 2. Early (1-3 Days Post-Op)

| Complication          | Cause                                  | Presentation                       | Management                               |
| --------------------- | -------------------------------------- | ---------------------------------- | ---------------------------------------- |
| **Atelectasis**       | Mucus plugging, shallow breathing      | Fever, tachypnea                   | Incentive spirometry, early mobilization |
| **Urinary Retention** | Post-op pain, spinal anesthesia        | Inability to void, suprapubic pain | Bladder scan, catheterization if needed  |
| **Paralytic Ileus**   | Anesthesia, opioid use, bowel handling | Absent bowel sounds, distension    | NPO, IV fluids, NG tube if severe        |
| **Wound Hematoma**    | Poor hemostasis, anticoagulation       | Swelling, discoloration            | Compression, evacuation if large         |

## 3. Intermediate (4-7 Days Post-Op)

| Complication                   | Cause                                | Presentation                        | Management                        |
| ------------------------------ | ------------------------------------ | ----------------------------------- | --------------------------------- |
| **Wound Infection**            | Staph aureus, poor asepsis           | Erythema, warmth, purulent drainage | Drainage, wound care, antibiotics |
| **Deep Vein Thrombosis (DVT)** | Immobility, hypercoagulability       | Leg pain, swelling                  | Anticoagulation (LMWH)            |
| **Pneumonia**                  | Aspiration, atelectasis              | Cough, fever, infiltrates on CXR    | Antibiotics, pulmonary hygiene    |
| **Anastomotic Leak**           | Intra-abdominal surgery complication | Fever, peritonitis, sepsis          | CT scan, re-exploration           |

## 4. Late (>7 Days Post-Op)

| Complication                      | Cause                         | Presentation                        | Management                               |
| --------------------------------- | ----------------------------- | ----------------------------------- | ---------------------------------------- |
| **Wound Dehiscence**              | Poor wound healing, infection | Open wound, serosanguinous drainage | Dressing, possible re-suturing           |
| **Incisional Hernia**             | Weak abdominal wall closure   | Bulging at incision site            | Surgical repair if symptomatic           |
| **Fistula Formation**             | Breakdown of anastomosis      | Persistent drainage, infection      | Conservative or surgical management      |
| **Adhesions & Bowel Obstruction** | Post-op fibrosis              | Colicky pain, distension            | NPO, NG decompression, surgery if needed |

## Prevention Strategies

- **Pre-op:** Optimize comorbidities, thromboprophylaxis.
- **Intra-op:** Aseptic technique, minimize blood loss.
- **Post-op:** Early mobilization, wound care, infection control.

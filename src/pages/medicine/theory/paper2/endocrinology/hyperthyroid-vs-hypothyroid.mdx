# Hyperthyroid vs Hypothyroid

| Feature                    | Hyperthyroidism                                           | Hypothyroidism                                      |
| -------------------------- | --------------------------------------------------------- | --------------------------------------------------- |
| **Definition**             | Excessive thyroid hormone production                      | Insufficient thyroid hormone production             |
| **Thyroid Hormone Levels** | ↑ T3, ↑ T4, ↓ TSH (in primary hyperthyroidism)            | ↓ T3, ↓ T4, ↑ TSH (in primary hypothyroidism)       |
| **Metabolic Rate**         | Increased (hypermetabolic state)                          | Decreased (hypometabolic state)                     |
| **Common Causes**          | - Graves' disease (autoimmune)                            | - Hashimoto's thyroiditis (autoimmune)              |
|                            | - Toxic multinodular goiter                               | - Iodine deficiency                                 |
|                            | - Thyroid adenoma                                         | - Post-thyroidectomy or radioiodine therapy         |
| **Symptoms**               | - Weight loss despite increased appetite                  | - Weight gain despite normal/reduced appetite       |
|                            | - Heat intolerance, sweating                              | - Cold intolerance                                  |
|                            | - Tachycardia (increased heart rate)                      | - Bradycardia (slow heart rate)                     |
|                            | - Nervousness, anxiety, irritability                      | - Depression, fatigue                               |
|                            | - Tremors, hyperreflexia                                  | - Muscle weakness, sluggish reflexes                |
|                            | - Diarrhea                                                | - Constipation                                      |
|                            | - Exophthalmos (bulging eyes, in Graves' disease)         | - Periorbital puffiness                             |
| **Skin and Hair**          | - Warm, moist skin                                        | - Dry, coarse skin                                  |
|                            | - Fine, thinning hair                                     | - Hair loss, brittle nails                          |
| **Goiter**                 | Present in Graves' disease or toxic goiter                | Present in Hashimoto's or iodine deficiency         |
| **Menstrual Changes**      | Oligomenorrhea (infrequent menstruation)                  | Menorrhagia (heavy menstruation) or amenorrhea      |
| **Complications**          | - Thyroid storm (life-threatening)                        | - Myxedema coma (life-threatening)                  |
|                            | - Atrial fibrillation                                     | - Heart failure, infertility                        |
| **Treatment**              | - Antithyroid drugs (e.g., methimazole, propylthiouracil) | - Thyroid hormone replacement (e.g., levothyroxine) |
|                            | - Radioactive iodine therapy                              | - Address underlying cause                          |
|                            | - Surgery (thyroidectomy)                                 |                                                     |

# ECG

## How to Count Heart Rate

> One small square = 0.04 s<br/>One large square = 0.2 s<br/>5 large squares = 1 s

### In Regular Rhythm (Sinus Rhythm)

> Heart rate = 300 / Number of large squares between 2 consecutive R wave

![sinus-rhythm](/medicine/ecg/sinus-rhythm.png)

### In Fast Rhythm (Tachycardia)

> Heart rate = 1500 / Number of small squares between 2 consecutive R wave

![SVT](/medicine/ecg/supraventricular-tachycardia.png)

### In Irregular Rhythm (Atrial Fibrillation)

> Heart rate = Number of R waves over a 10 second period x 6

![AF](/medicine/ecg/atrial-fibrillation.png)

## 12-Lead ECG

### Heart Walls, ECG Leads, and Coronary Artery Supply

> aVR normally shows negative waves.

![ECG Leads and Heart Walls](/medicine/ecg/ecg-leads-heart-walls.webp)

| Walls          | ECG Lead     | Coronary Artery Supply |
| :------------- | :----------- | :--------------------- |
| Lateral        | I, aVL, -aVR | LCx                    |
| Inferior       | II, III, aVF | RCA                    |
| Septal         | V1, V2       | LAD                    |
| Anterior       | V3, V4       | LAD                    |
| Anteriolateral | V5, V6       | LAD + LCx              |

- LCx = Left Circumflex Artery
- RCA = Right Coronary Artery
- LAD = Left Anterior Descending Artery

## Myocardial Infarction

### ECG Reciprocal Leads (Opposite Leads)

| Infarction                             | Leads with ST Elevation      | Reciprocal Leads (ST Depression) |
| -------------------------------------- | ---------------------------- | -------------------------------- |
| **Lateral** (LCx, diagonal branch LAD) | I, aVL, V5, V6               | II, III, aVF                     |
| **Inferior** (RCA, LCx)                | II, III, aVF                 | I, aVL, V1, V2                   |
| **Septal** (LAD)                       | V1, V2                       | V5, V6                           |
| **Anterior** (LAD)                     | V1, V2, V3, V4               | II, III, aVF                     |
| **Posterior** (RCA, LCx)               | V7, V8, V9 (posterior leads) | V1, V2                           |
| **Right Ventricular** (RCA)            | V4R, V5R                     | V1, V2, III, aVF                 |

## Axis Deviation

![Axis](/medicine/ecg/axis.png)

| Axis Deviation           | Lead I     | Lead aVF   |
| ------------------------ | ---------- | ---------- |

| **Left Axis Deviation**  | Positive   | _Negative_ |
| **Right Axis Deviation** | _Negative_ | Positive   |
| **Extreme Axis**         | _Negative_ | _Negative_ |
